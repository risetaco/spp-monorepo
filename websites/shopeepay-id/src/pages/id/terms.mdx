2.2.7 BI-FAST

(i) Metode BI-FAST diproses dengan menggunakan Proxy Address (sebagaimana didefinisikan di bawah ini).

(ii) "Proxy Address" adalah data alias yang digunakan untuk menggantikan identitas penerima dalam transaksi menggunakan BI-FAST.
